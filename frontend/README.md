# ⚛️ Frontend – Eventura

This is the frontend of **Eventura**, built with the latest technologies like **React 19**, **Vite**, **GSAP**, and **React Router v7**.

---

## 🛠 Tech Stack

- **React 19**
- **Vite**
- **GSAP** (for animations)
- **React Router v7**
- **ESLint** for linting

---

## 🚀 Getting Started

### 1. Install dependencies
- npm install

### 2. Run the app locally
- npm run dev

### 3. Start the backend
- npm start
